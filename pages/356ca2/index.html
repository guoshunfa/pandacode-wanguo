扫田技巧篇

# 扫田技巧篇

# 扫田相关概念描述

# 种田队伍描述

种田分为两种：普通种田队伍、武装种田队伍。

普通种田队伍是采集将搭配攻城车进行种田，属于普遍的种田队伍。

武装种田队伍是采集将或其它将搭配高级兵进行种田，高度伪装种田以至于会对扫田进行一定的阻碍。

# 狂热状态描述

什么叫狂热状态，在攻击或侦查敌方基地、资源点、旗帜、要塞时会触发一个buff（狂热），狂热状态下有攻击加成，时间为7分钟，但是在狂热状态期间不能进行迁城，这点必须注意。也就是说扫完田之后，我们需要冷静7分钟才能战术转移，在这7分钟时间内尽量不要被敌方发现。

# 集结描述

正常情况下我们在攻击野蛮人要塞或地方基地时会进行集结进攻，如果被集结的是敌方玩家基地，就算他迁城也是会被集结。一般锁定敌方位置的技巧是，1兵集结敌方单位8个小时，可以一直跟踪敌方单位动向，不管他如何迁城都会被我们知道。这一点技巧也会被敌方进行掌握，如果自己被锁定了，那我们能做的就只有祈祷🙏了。

# 逃荒式扫田技巧指导

逃荒式扫田是下下策，只有急眼了才会使用这个策略。

分步进行：

  1. 选择种田队伍多的地方，然后迁城。（也可以随机迁城。）
  2. 快乐的进行扫田。
  3. 扫田结束后，等待狂热状态结束进行迁城。
  4. 休息或继续进行第一步。（结束，但也是个开始。😊）
